# Самообучающийся 2D-гонщик Формулы 1 с NEAT

Проект симуляции самообучающегося автомобиля, который с помощью генетических алгоритмов (NEAT) учится проезжать трассу без столкновений.

## 🚀 Функциональность

- **Самообучение**: Автомобили эволюционируют через NEAT (NeuroEvolution of Augmenting Topologies)
- **Зрение**: 7 лучей (raycasting) для обнаружения границ трассы
- **Физика**: Реалистичная физика автомобиля (инерция, ускорение, торможение, поворот)
- **Визуализация**: Отображение трассы, автомобилей, лучей и статистики
- **Генерация трассы**: Программная генерация овальной или извилистой трассы

## 🎮 Использование

### 1. Обучение нейросети

Запустите программу и выберите режим **1**:

```bash
python main.py
```

В процессе обучения:
- Каждое поколение состоит из 50 автомобилей
- Автомобили учатся ехать по трассе, избегая столкновений
- Fitness (приспособленность) растет с пройденным расстоянием и временем жизни
- По умолчанию: 30 поколений (можно изменить в коде)

После обучения лучший геном сохранится в файл `best_genome.pkl`.

### 2. Визуализация лучшего автомобиля

Запустите программу снова и выберите режим **2**:

```bash
python main.py
```

Вы увидите, как лучший из всех эволюционировавших автомобилей проезжает трассу.

## 🔧 Настройка параметров

### Конфигурация NEAT (`config-feedforward`)

Основные параметры для настройки обучения:

```
pop_size = 50          # Размер популяции (больше = дольше обучение, но лучше результат)
num_inputs = 8         # Количество входов (7 датчиков + скорость + угол)
num_hidden = 6         # Количество нейронов в скрытом слое
num_outputs = 3        # Выходы: [газ, поворот_влево, поворот_вправо]
```

### Параметры автомобиля (`car.py`)

```python
# Физика
self.max_speed = 8.0           # Максимальная скорость
self.acceleration = 0.15       # Ускорение
self.braking = 0.2             # Торможение
self.turn_speed = 4.0          # Скорость поворота (градусов/кадр)
self.friction = 0.05           # Трение

# Датчики
self.num_sensors = 7           # Количество лучей
self.sensor_length = 150       # Длина лучей (пикселей)
self.sensor_angles = [-90, -45, -20, 0, 20, 45, 90]  # Углы лучей
```

### Настройка скорости обучения

В `main.py` измените:

```python
# В evaluate_genomes:
max_time = 1000  # Максимальное время жизни в кадрах (больше = дольше обучение)

# В run_neat:
winner = self.population.run(self.evaluate_genomes, num_generations=30)
# ↑ увеличьте для более качественного обучения
```

## 🎯 Как работает нейросеть

### Входы (Inputs):
1. `sensor_0` - расстояние до стены слева (-90°)
2. `sensor_1` - расстояние до стены слева-сбоку (-45°)
3. `sensor_2` - расстояние до стены слева-вперед (-20°)
4. `sensor_3` - расстояние прямо (0°)
5. `sensor_4` - расстояние до стены справа-вперед (+20°)
6. `sensor_5` - расстояние до стены справа-сбоку (+45°)
7. `sensor_6` - расстояние до стены справа (+90°)
8. `speed` - нормализованная скорость (0.0 - 1.0)
9. `angle` - нормализованный угол поворота

### Выходы (Outputs):
1. `gas` > 0.5 - ускоряться, < 0.3 - тормозить
2. `left` > `right` - поворот влево
3. `right` > `left` - поворот вправо

## 🛠️ Решение проблем

### Автомобили не учатся (все сразу умирают)
- **Увеличьте длину лучей**: `self.sensor_length = 200`
- **Уменьшите скорость**: `self.max_speed = 5.0`
- **Увеличьте время жизни**: `max_time = 2000`
- **Измените стартовую позицию**: переместите `start_x, start_y` внутрь трассы

### Автомобили едут по кругу и не поворачивают
- **Увеличьте `turn_speed`**: `self.turn_speed = 6.0`
- **Уменьшите `num_hidden`** в конфиге: попробуйте 4 вместо 6
- **Попробуйте другую трассу**: используйте `generate_winding_track()`

### Обучение слишком медленное
- **Уменьшите `pop_size`**: 30 вместо 50
- **Уменьшите `num_generations`**: 20 вместо 30
- **Упростите трассу**: используйте овальную трассу

### Автомобили выезжают за границы
- Проверьте, что стартовая позиция находится **внутри** белой области трассы
- Увеличьте `sensor_length`, чтобы автомобили видели стену заранее

## 📊 Интерпретация fitness

Fitness считается по формуле:
```
fitness = пройденное_расстояние + время_жизни * 10 + скорость * 5 - 1000(если умер)
```

Чем больше fitness, тем лучше автомобиль.

## 🎨 Трасса

Трасса генерируется программно:
- **Белые пиксели** = проезжая часть (трасса)
- **Черные пиксели** = стены/препятствия

Маска трассы используется для:
1. Raycasting (лучи проверяют цвет пикселя)
2. Collision detection (столкновение с черным = смерть)

## 📈 Мониторинг обучения

В процессе обучения NEAT выводит:
- Номер поколения
- Лучший fitness текущего поколения
- Средний fitness
- Размеры видов (species)
